# IG Browser Scripts

Simple browser console scripts for Instagram Web.

## Features

- Find users who don't follow you back
- Automatic scrolling
- Clickable Instagram profile links
- Clipboard export
- No extensions required
- No external API usage

---

# Usage

1. Open Instagram Web
2. Go to your profile
3. Press `F12`
4. Open the `Console` tab
5. Paste `allow pasting`
6. Paste the script
7. Press `Enter`

The script will automatically:

- Open Following
- Scroll until fully loaded
- Open Followers
- Compare lists
- Print users that don't follow back

---

# Notes

- Instagram changes UI frequently
- Some selectors may break after updates
- Large accounts may take longer to load
- Works best on desktop Chrome
